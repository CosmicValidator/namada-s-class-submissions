## Namada Indexer API

## Usage
### get stats
https://namada-se-smart-api.cosmicvalidator.com/stats

### get blocks info
https://namada-se-smart-api.cosmicvalidator.com/blocks?page=1&page_size=10

### get txs 
https://namada-se-smart-api.cosmicvalidator.com/txs?page=1&page_size=10

### get proposals
https://namada-se-smart-api.cosmicvalidator.com/proposals?page=1&page_size=10

### get tx by hash
https://namada-se-smart-api.cosmicvalidator.com/tx/{tx_hash}

### get tx by memo
https://namada-se-smart-api.cosmicvalidator.com/tx_by_memo/{memo}
